Predicting pEC50 for Dopamine
Receptor Ligands

Personal Project  ·  RDKit · ChEMBL · TensorFlow · Scikit-learn

Dopamine receptors are critical therapeutic targets for neurological and psychiatric conditions — from Parkinson's to schizophrenia. This project forecasts pEC50 values (the logarithmic concentration needed for half-maximal ligand effect), bridging my PhD-level chemistry expertise with production ML pipelines. Data was compiled from ChEMBL, enriched with molecular descriptors via RDKit, and fed into both classical ML and deep learning models.

RDKitChEMBLTensorFlow Scikit-learnXGBoostGradient Boosting keras_tunerMolecular FingerprintsFeature Engineering

Background

pEC50 quantifies binding potency — a critical number in drug discovery. Predicting it computationally from molecular structure allows researchers to screen thousands of candidate compounds before any lab synthesis, dramatically narrowing the search space for effective therapeutics.

Methodology

From ChEMBL to Predictions

01

Custom Dataset Creation

Extracted SMILES strings and EC50 measurements for all five dopamine receptor subtypes (D1–D5) using the ChEMBL WebResource client. Supplemented with molecular descriptors (MW, LogP, TPSA, rotatable bonds) computed via RDKit.

02

Exploratory Data Analysis

Visualised feature relationships and pEC50 distributions across all receptor subtypes. Observed meaningful subtype variation, confirming the dataset was suitable for predictive modelling.

03

Classical ML — Ensemble Methods

Tested Random Forest, Gradient Boosting, and XGBoost. Applied both Random Search and Grid Search hyperparameter optimisation. Evaluated on MAE, RMSE, and R².

04

Deep Learning Extension — ANNs

Built Artificial Neural Networks with TensorFlow/Keras. Used keras_tuner for automated architecture search. Compared ANN performance against best classical ML baseline.

Benchmark Context

A validation MAE of 0.60 aligns with published benchmarks in the range of 0.52–0.61 reported in Scientific Reports for comparable dopamine receptor datasets — confirming the model's practical relevance.

"Top contributing features: LogP, molecular weight, TPSA, and rotatable bonds — descriptors that encode the 3D structure and pharmacokinetic profile of each molecule."

Key Observations & Limitations

Gradient Boosting maintained superior performance throughout, despite notable overfitting even with early-stopping. The deep learning extension reduced overfitting but did not improve accuracy — a common finding when dataset size is the limiting factor.

Future direction: Graph Neural Networks (GNNs) represent the natural next step, as they can directly encode molecular topology rather than relying on hand-crafted descriptors.
